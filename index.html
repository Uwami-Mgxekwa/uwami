Wrote my first "Hello World" in C++ and got completely hooked. Spent countless hours figuring out 
                           pointers and wondering why my programs crashed so much. Those early struggles taught me more 
                           about persistence than any textbook ever could.

                        

                            C++
                            Algorithms
                        

                    

                

                

                    

                        
Finding My Language

                        
Discovered Java and fell in love with Swing. Built my first GUI application and felt like a wizard. 
                           There's something deeply satisfying about creating interfaces that people can actually click and use. 
                           Java became my home base.

                        

                            Java
                            Swing
                            OOP
                        

                    

                

                

                    

                        
Web Curiosity

                        
Decided to explore the web side of things. HTML and CSS felt like learning a new language,

                            CSS3

                            SQL

                            Data Analysis
